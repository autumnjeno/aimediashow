# AI x 媒体未来 — 首页设计文档

## 概述

"AI x 媒体未来"网站首页，展示 AI 如何重塑媒体行业。采用**叙事+数据混合**方案，深色科技风视觉风格。

## 技术栈

- 纯 HTML + CSS 单页面，无框架依赖
- 轻量 CSS 动画（transition、hover 效果），不引入动画库
- 响应式设计（桌面端优先，移动端适配）

## 视觉规范

### 配色

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | `#050810` | 页面底色 |
| 卡片背景 | `#111833` | 功能卡片、数据条背景 |
| 主题色 1 | `#00d4ff` (cyan) | AI内容生产区块、导航高亮 |
| 主题色 2 | `#7c3aed` (purple) | 智能分发区块 |
| 主题色 3 | `#f59e0b` (gold) | 多媒体处理区块 |
| 主题色 4 | `#10b981` (green) | 数据洞察区块 |
| 主文字 | `#fff` | 标题 |
| 副文字 | `#8899bb` | 描述文字 |
| 辅助文字 | `#667` / `#445` | 数据标签、版权 |

### 通用样式

- 圆角：卡片 12px，按钮 24px（胶囊形），小元素 6-8px
- 边框：半透明主题色，如 `rgba(0,212,255,0.15)`
- 阴影：CTA 主按钮带发光阴影 `box-shadow: 0 0 20px rgba(0,212,255,0.3)`
- 渐变：标题用 `linear-gradient(90deg, #00d4ff, #7c3aed)` + `background-clip: text`

## 页面结构

### 1. 导航栏

固定顶部，半透明深色背景 `rgba(5,8,16,0.9)`。

- **左侧**：Logo 图标（渐变圆角方块 + 文字"AI x 媒体未来"）
- **中间**：导航链接 — 首页 / AI内容生产 / 智能分发 / 多媒体处理 / 数据洞察
- **右侧**：CTA 按钮"联系我们"（渐变胶囊形）

### 2. Hero 区

全屏高度，居中布局。

- **背景**：深色底 + 三色辐射光晕（cyan/purple/gold 位置不同）+ 半透明网格线（40px 间距）
- **顶部标签**：胶囊形边框标签 "2026 · 媒体行业AI转型指南"
- **主标题**："当AI重新定义 / 新闻的生产与传播"，"新闻的生产与传播"用 cyan→purple 渐变
- **副标题**：描述 AI 在各环节重塑媒体
- **双 CTA 按钮**：
  - 主按钮"探索AI如何改变媒体"（渐变背景+发光阴影）
  - 副按钮"下载行业报告"（透明边框）
- **底部数据指标条**：4 列等宽，显示关键数据
  - 78% 媒体机构已采纳AI
  - 3.2× 内容生产效率提升
  - 96% 分发精准度
  - 50% 运营成本降低
- **滚动提示**：底部 "↓ 向下滚动探索"，pulse 动画

### 3. 场景区块（×4）

4 个场景区块采用统一模板，居中卡片网格布局，仅切换主题色。

每个区块包含三层：

**第一层 — 标题区**（居中）：
- 顶部装饰线（主题色渐变）
- 图标 + 场景标签（大写字母间距）
- 主标题（关键词用主题色高亮）
- 描述文字

**第二层 — 功能卡片**（3 列居中）：
- 每个卡片：图标 + 标题 + 描述
- 卡片背景：主题色 5% 透明度
- 卡片边框：主题色 15% 透明度
- hover 效果：边框加亮 + 轻微上移

**第三层 — 数据亮点**（横排指标条）：
- 3 个数据指标横排
- 数字用主题色，大号粗体
- 标签文字 `#667`

4 个场景的具体内容：

#### 3.1 AI内容生产（cyan #00d4ff）

- 标题：让AI成为你的**首席编辑**
- 功能卡片：智能撰稿 / 选题推荐 / 质量审核
- 数据：3.2× 生产效率提升 / 85% 人工干预减少 / 24/7 全天候不间断

#### 3.2 智能分发（purple #7c3aed）

- 标题：每个读者都是**唯一受众**
- 功能卡片：个性推荐 / 多渠道推送 / 实时优化
- 数据：96% 分发精准度 / 3.2× 触达效率提升 / 50% 用户留存增长

#### 3.3 多媒体处理（gold #f59e0b）

- 标题：文字、图片、视频**一键生成**
- 功能卡片：AI剪辑 / 智能标注 / 跨媒体转换
- 数据：5min 视频生成 / 10× 处理效率 / 100+ 格式支持

#### 3.4 数据洞察（green #10b981）

- 标题：用数据驱动每一个**决策**
- 功能卡片：舆情监测 / 行为分析 / 效果归因
- 数据：360° 全景洞察 / 实时数据更新 / 99.9% 系统可用性

### 4. CTA 区

- **背景**：底部辐射光晕 + 网格线（与 Hero 区呼应）
- **装饰线**：居中渐变线
- **主标题**："准备好拥抱**AI驱动的媒体未来**了吗？"（渐变文字）
- **副标题**："加入数百家媒体机构的行列，开启您的AI转型之旅"
- **双按钮**：
  - 主按钮"开始免费体验"（渐变+发光）
  - 副按钮"预约演示"（透明边框）

### 5. Footer

单行三段式布局，`1px` 半透明分隔线与 CTA 区隔开。

- **左侧**：Logo（同导航栏）
- **中间**：导航链接 — 首页 / 关于我们 / 联系方式 / 隐私政策
- **右侧**：版权信息 "© 2026 AI x 媒体未来. All rights reserved."

## 交互动效

- **hover 效果**：功能卡片边框加亮 + 轻微上移（`transform: translateY(-4px)`，`transition: all 0.3s`）
- **导航链接**：hover 变色为主题色
- **按钮**：hover 阴影加深
- **滚动提示**：`animation: pulse 2s infinite`
- **数据数字**：无滚动触发动画，静态展示

## 文件结构

```
index.html        — 单页面文件
styles.css        — 样式文件（或内联于 index.html）
```

## 响应式断点

- 桌面端（>1024px）：完整布局
- 平板端（768-1024px）：功能卡片从 3 列变 2 列
- 移动端（<768px）：单列堆叠，导航折叠为汉堡菜单
